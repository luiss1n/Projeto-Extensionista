# Projeto-Extensionista
Projeto extensionista voltado ao desenvolvimento de um sistema próprio para gestão de pedidos de lanchonete. Criado do zero, o aplicativo encontra-se em fase de protótipo, permitindo a anotação manual de pedidos. A próxima etapa prevê a integração com API para leitura automática de mensagens, ampliando eficiência e usabilidade.
