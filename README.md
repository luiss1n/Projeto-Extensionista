# Projeto-Extensionista
Este projeto extensionista tem como objetivo o desenvolvimento de um sistema próprio, criado integralmente do zero, voltado para a otimização do processo de registro manual de pedidos em lanchonetes. Atualmente, o projeto encontra-se em fase de protótipo, por meio de um aplicativo que permite ao atendente organizar, registrar e gerenciar pedidos de forma simples e eficiente.

A proposta busca modernizar um processo tradicionalmente manual, oferecendo uma solução acessível, intuitiva e adaptada às necessidades reais de pequenos estabelecimentos. Além disso, está em andamento a implementação de uma integração via API para leitura e interpretação automática de mensagens, permitindo que pedidos enviados por canais digitais sejam identificados e convertidos diretamente para o sistema. Essa funcionalidade ampliará a eficiência do atendimento e reduzirá erros operacionais.

O projeto se destaca pela construção completa da solução, desde sua concepção até a fase atual de testes, evidenciando o desenvolvimento de habilidades práticas em programação, análise de requisitos e prototipagem de software. A próxima etapa inclui validação com usuários reais, refinamento da interface e integração dos módulos automatizados.
